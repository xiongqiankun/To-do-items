# 待办事项应用

一个现代化的待办事项管理应用，采用 Vue 3 + Tailwind CSS + Pinia 技术栈开发。

## 功能特性

### 核心功能
- ✅ 任务管理：创建、编辑、删除任务
- ✅ 任务分类：支持自定义分类和颜色
- ✅ 优先级设置：高、中、低三个优先级
- ✅ 截止时间：自动设置为当前时间的一天后
- ✅ 任务状态：标记任务为已完成或未完成
- ✅ 数据持久化：使用 localStorage 存储数据
- ✅ 深色模式：支持明暗主题切换

### 高级功能
- 🔍 任务搜索：快速查找任务
- 📊 数据统计：任务完成率、分类分布等
- 📱 响应式设计：适配各种屏幕尺寸
- 💾 数据导入导出：备份和恢复数据
- 🎨 现代界面：优雅的视觉设计和流畅的动画效果

## 技术栈

- **前端框架**：Vue 3 + Composition API
- **状态管理**：Pinia
- **样式**：Tailwind CSS v3
- **构建工具**：Vite
- **数据可视化**：Chart.js
- **日期处理**：Day.js

## 核心功能使用指南

### 创建任务
1. 点击页面顶部的"新建任务"按钮
2. 填写任务标题、描述、分类、优先级和截止时间
3. 点击"创建任务"按钮保存

### 管理任务
- **编辑任务**：点击任务卡片上的编辑图标
- **删除任务**：点击任务卡片上的删除图标
- **标记完成**：点击任务卡片左侧的圆形复选框

### 筛选和排序
- 使用顶部的筛选栏按分类、优先级和状态筛选任务
- 使用排序下拉菜单按创建时间、截止时间或优先级排序
- 使用搜索框快速查找任务

### 查看统计
- 点击导航栏的"统计"选项
- 查看任务完成率、分类分布和优先级分布
- 查看各分类的详细完成情况

### 自定义设置
- 点击导航栏的"设置"选项
- 切换深色/浅色模式
- 管理任务分类（添加、编辑、删除）
- 导出/导入任务数据
